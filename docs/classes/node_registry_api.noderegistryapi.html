Class NodeRegistryApi

NodeRegistryApi instance is heart of the engine. It simply provides an interface between Web3 instance and the rest of engine. The Web3 instance can be created using any provider including In3's web3-provider. NodeRegistryApi instance also need address of the NodeRegistryLogic contract to interact with by sending transactions and calling functions.

    Parameters

getNodesCount

  • getNodesCount(blockNumber?: bigint): Promise<number>
    • Defined in node-registry-api.ts:93

    Makes a request to 'totalNodes():uint64' function of logic contract. The result specifies number of currently registered nodes (In3 servers). If a blockNumber parameter specified, then makes sure the result must specify the node number in block at 'blockNumber'. Specifying blockNumber is a requirement when doing recurring requests based on the result.
